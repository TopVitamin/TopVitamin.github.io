浅谈产品数据分析

发表于 2017-08-15 | 分类于 产品

前言

数据分析 = 数据 + 分析，身为一个产品经理要学会如何数据分析，既然要学那么我们得先了解下这些数据有哪些，又应该如何进行分析？

数据

我们需要关注数据来源于三个大分类：

  1. 自身产品数据
  2. 竞品的数据
  3. 行业大环境的数据

关于自身数据

自身的数据有：

  1. 用户数据
  2. 市场推广营销数据
  3. 产品功能数据
  4. 运营数据
  5. 产品内容数据

1.用户数据

基础属性：

日活跃用户数：每天大概有多少活跃用户，流量有多少
累计用户数：用户的总数有多少，能达到多少
流失用户数：一次性用户，丢失的原因是什么
每天新增用户数：关注新增用户，知道用什么办法什么时间段能吸引更多的用户
投诉用户数：知道自己产品的不足有多少，如果很多人反馈同一个问题就需要特别重视了
用户年龄段：年龄决定了思维方式和消费方式
用户性别：性别和消费习惯有关系，也和产品的使用体验度有关系（女生不会用太麻烦的东西）
用户地区分布：地区可以更加准确的分析用户，因地制宜
用户群体：学生还是白领，家庭主妇还是年轻大学生，找到最适合的营销方式

拓展属性

忠实用户：脑残粉，忠实的用户，基数最小
订单用户：下过单还会继续下单的用户
回访用户：上次用觉得不过，再次回访
浏览用户：看一眼不够，可能还停留试用一下
跳出用户：看一下就走，基数最大，最底层

2. 市场推广营销数据

市场基础数据

  • 用户从哪里来
  • 过来的用户的质量和数量怎么样
  • 吸引用户的成本是多少
  • 吸引用户的方法有多少
  • …

指标数据

  • 曝光数
  • 点击数
  • 新访客数量
  • 新注册用户数
  • 新付费用户数

  • 客单价分层次的后的销售金额

  • 跳出率

  • 访问时长
  • 浏览次数
  • 访问深度
  • 转化率
  • 留存率
  • 访问频次/间隔
  • …

3. 产品功能数据

  • 哪些功能用户使用的较多
  • 留存与哪些功能相关性较大
  • 被吐槽的功能有哪些
  • 可优化的功能有哪些，哪些功能是可以显著提升留存率，成交率等
  • …

4. 运营数据

  • 销售额
  • UV
  • 客单价
  • 转化率
  • 新用户数
  • 毛利率
  • 用户回购率
  • …

5. 产品内容数据

  • 视频的播放量
  • 产品的访问量
  • 用户评论数
  • 分享次数
  • PV和一些其他内容性指标
  • …

竞品和行业数据

  1. 市场规模
    行业的规模有多大，这一块蛋糕大概是有多大的，然后自己位置是怎么样的

  2. 市场推广力度
    市场的推广力度和品牌口碑类似，推广强度和力度有多大，数据可以参考百度 指数等

  3. 品牌口碑
    品牌口碑主要是指用户口碑，可以做一些简单的用户访谈和一些网上言论的统计，当然这个只能是大概的参考

  4. 行业趋势
    了解整个行业的动态，热点和趋势，帮助自己对产品有一个清晰的认识和定位

  5. 人群属性

了解该行业的用户的人群属性，地域分布和年龄段等
